# Product Specification

- The product is a simple proof of concept "wallet aggregator".
- The product is a webapp written in typescript with a vite frontend and tailwind styling
- Styling is absolutely minimal, clear, light accents. Well structured, trustworthy.
- The product has the following functionality:
-- User can generate an Ethereum wallet (i.e. a private key is generated etc)
-- User can connect Metamask and use it to make transactions
-- User can import private keys in QR format (via camera) or import a private key string directly
-- User can import Ethereum address
-- The user can make transfers for all wallet types (except view only)
-- The user see a transparent list of crypto holdings for all wallets
-- The web app provides a small local ETH testnet that the webapp can connect to for simple testing
-- The app has a predefined wallet which is already filled with 100ETH in the test net.
- The app is a single page app exclusively
- The app is POC only, it does not have to be secure etc.
